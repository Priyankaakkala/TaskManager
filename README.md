# Task Manager

Full-stack Task Manager (Node/Express backend, React frontend) with JWT auth and MongoDB.

## Run (development)

Backend:

```bash
cd backend
npm install
# set MONGO_URI in .env
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm start
```

## API (example)
- POST /api/auth/register
- POST /api/auth/login
- GET/POST/PUT/DELETE /api/tasks
